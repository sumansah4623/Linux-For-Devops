**************Command for creating Group***************

*************Steps for creating groups****************

1. sudo groupadd <groupName>  (creating new group)
2. cat /etc/group   (display all group on screen)
3. sudo gpasswd -a <userName> <groupName>    (adding user to group)
4. sudo gpasswd -m <user1>, <user2>, <user3> <GroupName> (adding multiple users to one group)
5. cat /etc/group    (now check)
6. sudo groupdel <groupName> (for deleting group)
